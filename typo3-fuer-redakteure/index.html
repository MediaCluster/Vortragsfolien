Anmeldung am Backend

  • http://typo3.mediacluster.de/typo3/

Module

  • Web
    • Seite: Inhalte bearbeiten
    • Anzeigen: Vorschau im Backend
    • Liste: Datensätze verwalten (Adressen, Benutzer, Termine, usw.)
    • Workspaces: Arbeitsumgebungen (z.B. Live und Entwurf)
    • Info: verschiedene Info-Module (z.B. Linkvalidierung)
    • Zugriff: Benutzer- und Gruppenrechte
    • Funktionen: Assistenten (z.B. mehrere Seiten anlegen)
    • Nachrichten: News-Verwaltung
  • Datei
    • Dateiliste: Einfache Dateiverwaltung
  • Benutzerwerkzeuge
    • Aufgaben: Aufgabenverwaltung
    • Einstellungen: Persönliches Profil verwalten

Seitenbaum

  • Seiten können per Drag 'n' Drop verschoben werden
  • Häufig verwendete Aktionen sind auch über ein
    Kontextmenü erreichbar

Inhalte (Seitenansicht)

Wenn das Seitenmodul "TemplaVoila!" (grünes TV) verwendet wird,
können auch Inhaltselemente per Drag 'n' Drop verschoben werden.

Inhalte hinzufügen

Neue Inhalte sollten immer über das Seiten-Modul hinzugefügt werden, da sie sonst keiner "Spalte" zugeordnet und somit nicht angezeigt werden.

Inhalte (Listenansicht)

Dateiverwaltung (mit Freigaben)

  • Häufig verwendete Aktionen sind auch über ein
    Kontextmenü erreichbar
  • Wenn "Freigaben" definiert und zugewiesen wurden, können Redakteure nur diese Verzeichnisse sehen und pflegen.
  • Administratoren haben direkten Zugriff auf das Dateisystem und die Verzeichnisstruktur (ohne Einschränkungen).
